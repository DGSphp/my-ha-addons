# Bluetooth Manager

## Installation
1. Add the repository `https://github.com/DGSphp/my-ha-addons` to your Add-on Store.
2. Install **Bluetooth Manager**.
3. Start the add-on.
4. Click **Open Web UI** or look for "BT Manager" in your sidebar.

## Usage
- **Scan:** Click the scan button to update the list of devices seen by the dongle.
- **Connect/Disconnect:** Use the buttons next to the device name.

## Troubleshooting
If devices are not showing up, ensure your Bluetooth dongle is supported by Home Assistant OS and passed through correctly.
